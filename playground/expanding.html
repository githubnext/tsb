Expanding Windows

Growing-window aggregations — mirrors pandas.Series.expanding() and pandas.DataFrame.expanding().

1 — Basic expanding mean (cumulative average)

Call series.expanding() to get an Expanding object, then call .mean(). Each value is the average of all observations up to and including that position.

import { Series } from "tsb";

const temperatures = new Series({
  data: [22, 25, 20, 28, 24, 30, 18],
  name: "temp_c",
});

// Expanding mean — each value is the running average
const runningMean = temperatures.expanding().mean();
console.log(runningMean.toArray());
// [22, 23.5, 22.333..., 23.75, 23.8, 24.833..., 23.857...]

2 — Cumulative sum and running min/max

All standard aggregations are supported. sum() gives the cumulative sum; min() and max() track the historical extremes.

import { Series } from "tsb";

const scores = new Series({ data: [10, 25, 5, 30, 15, 20] });

console.log("Cumulative sum:", scores.expanding().sum().toArray());
console.log("Running min:",    scores.expanding().min().toArray());
console.log("Running max:",    scores.expanding().max().toArray());
▶ run

3 — minPeriods: require minimum observations

By default minPeriods = 1, so the first value is always computed. Raise it to require more observations before producing a result.

import { Series } from "tsb";

const s = new Series({ data: [10, 20, 30, 40, 50] });

// Default: minPeriods=1 — result from position 0
console.log("minPeriods=1:", s.expanding().sum().toArray());
// [10, 30, 60, 100, 150]

// minPeriods=3 — null until 3 observations are present
console.log("minPeriods=3:", s.expanding({ minPeriods: 3 }).sum().toArray());
// [null, null, 60, 100, 150]
▶ run

4 — Handling missing values

Null and NaN values are skipped during aggregation — only valid numeric observations contribute to each result.

import { Series } from "tsb";

const s = new Series({ data: [1, null, 3, null, 5] });

// Nulls are ignored; count() shows how many valid values have been seen
console.log("sum:   ", s.expanding().sum().toArray());   // [1, 1, 4, 4, 9]
console.log("count: ", s.expanding().count().toArray()); // [1, 1, 2, 2, 3]
console.log("mean:  ", s.expanding().mean().toArray());  // [1, 1, 2, 2, 3]
▶ run

5 — Standard deviation and variance

Expanding std() and var() use sample statistics (ddof=1) by default, matching pandas. The first element is always null since variance requires at least 2 observations.

import { Series } from "tsb";

const s = new Series({ data: [2, 4, 4, 4, 5, 5, 7, 9] });

const stdResult = s.expanding().std();
console.log("std:", stdResult.toArray().map(v => v !== null ? +v.toFixed(4) : null));
// [null, 1.4142, 1.1547, 1.0, 1.0954, 1.2111, 1.6036, 2.0]
▶ run

6 — Custom aggregation with apply()

Use .apply(fn) to run any aggregation over the expanding window. The function receives an array of valid numeric values.

import { Series } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });

// Expanding product (cumulative product)
const product = s.expanding().apply((nums) => nums.reduce((a, b) => a * b, 1));
console.log("product:", product.toArray()); // [1, 2, 6, 24, 120]

// Expanding range (max - min)
const range = s.expanding().apply((nums) => Math.max(...nums) - Math.min(...nums));
console.log("range:  ", range.toArray());   // [0, 1, 2, 3, 4]
▶ run

7 — DataFrame.expanding()

Expanding windows work on DataFrames too — aggregations are applied independently to each column.

import { DataFrame } from "tsb";

const df = DataFrame.fromColumns({
  revenue: [100, 150, 120, 200, 180],
  costs:   [ 60,  90,  70, 110,  95],
});

// Expanding mean per column — running average revenue and costs
const means = df.expanding().mean();
console.log("Revenue running avg:", means.col("revenue").toArray());
console.log("Costs running avg:  ", means.col("costs").toArray());

8 — Comparing expanding vs rolling

Expanding windows grow to include all past observations; rolling windows use a fixed size. Use rolling when you want a "recent" view; expanding when you want a "historical" view.

import { Series } from "tsb";

const prices = new Series({ data: [100, 105, 98, 112, 108, 115, 103] });

const rollingMean3 = prices.rolling(3).mean();
const expandingMean = prices.expanding().mean();

console.log("Rolling(3) mean:", rollingMean3.toArray().map(v => v !== null ? +v.toFixed(2) : null));
console.log("Expanding mean: ", expandingMean.toArray().map(v => v !== null ? +v.toFixed(2) : null));
