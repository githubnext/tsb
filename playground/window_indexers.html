🪟 Window Indexers — tsb

Custom window indexers let you define arbitrary window shapes for rolling computations. Mirrors pandas.api.indexers.

Classes: BaseIndexer · FixedForwardWindowIndexer · VariableOffsetWindowIndexer · applyIndexer()

1. FixedForwardWindowIndexer

The default pandas Rolling looks backward. FixedForwardWindowIndexer instead looks forward — each row's window covers the next N rows. 

import { FixedForwardWindowIndexer, applyIndexer } from "tsb";

const idx = new FixedForwardWindowIndexer({ windowSize: 3 });
const values = [10, 20, 30, 40, 50];
const result = applyIndexer(idx, values, nums => nums.reduce((a,b)=>a+b,0));
// → [60, 90, 120, 90, 50]

2. VariableOffsetWindowIndexer

Defines a different look-back (or look-forward) depth for each row. Useful for event-driven windows or irregular-frequency time series. 

import { VariableOffsetWindowIndexer, applyIndexer } from "tsb";

// Trailing window — look back by offsets[i] rows
const idx = new VariableOffsetWindowIndexer({ offsets: [0, 1, 2, 1, 0] });
const result = applyIndexer(idx, [10,20,30,40,50], nums => nums.reduce((a,b)=>a+b,0));
// → [10, 30, 60, 70, 50]

3. Custom BaseIndexer subclass

Subclass BaseIndexer to implement any window shape you need.

import { BaseIndexer, applyIndexer } from "tsb";

// Symmetric (centered) window
class CenteredIndexer extends BaseIndexer {
  constructor(private half: number) { super({ windowSize: 2 * half + 1 }); }
  getWindowBounds(n: number) {
    const start = new Int32Array(n);
    const end   = new Int32Array(n);
    for (let i = 0; i < n; i++) {
      start[i] = Math.max(0, i - this.half);
      end[i]   = Math.min(n, i + this.half + 1);
    }
    return [start, end];
  }
}

const idx = new CenteredIndexer(1); // window of [-1, 0, +1]
const result = applyIndexer(idx, [10,20,30,40,50], nums => nums.reduce((a,b)=>a+b,0)/nums.length);
// → [15, 20, 30, 40, 45]
