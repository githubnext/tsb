unique / nunique

Unique values and unique-value counts — mirrors pandas.unique(), Series.nunique(), and DataFrame.nunique().

Overview

Three functions for working with unique values:

  • unique(series) — returns a new Series of unique values in first-appearance order (hash-table based, does NOT sort).
  • nunique(series) — returns the count of distinct non-missing values.
  • dataFrameNunique(df) — returns a Series of per-column (or per-row) unique-value counts.
Missing values (null, undefined, NaN) are treated as a single distinct missing category. By default nunique and dataFrameNunique exclude missing values (dropna: true), while unique includes them (dropna: false) — matching pandas behaviour.

unique(series)

import { Series, unique } from "tsb";

const s = new Series([3, 1, 2, 1, 3, null, null]);
const u = unique(s);
console.log(u.values);          // [3, 1, 2, null]  — first-appearance order
console.log(u.values.length);   // 4

// Exclude missing values
const u2 = unique(s, { dropna: true });
console.log(u2.values);  // [3, 1, 2]
[3, 1, 2, null] length: 4 dropna=true: [3, 1, 2]

nunique(series)

import { Series, nunique } from "tsb";

const s = new Series([1, 2, 2, null, NaN]);
console.log(nunique(s));                    // 2  (null and NaN excluded)
console.log(nunique(s, { dropna: false })); // 3  (missing is one category)
dropna=true (default): 2 dropna=false: 3

String series

import { Series, nunique } from "tsb";

const cities = new Series(["Paris", "London", "Paris", "Berlin", "London"]);
console.log(nunique(cities)); // 3
3

dataFrameNunique(df) — axis=0 (default)

Count unique values per column (axis=0, the default).

import { DataFrame, dataFrameNunique } from "tsb";

const df = DataFrame.fromColumns({
  animal: ["cat", "dog", "cat", "bird"],
  legs:   [4, 4, 4, 2],
  alive:  [true, true, true, null],
});

const result = dataFrameNunique(df);
console.log(result.index.values); // ["animal", "legs", "alive"]
console.log(result.values);       // [3, 2, 1]  (null excluded from "alive")
columns: ["animal", "legs", "alive"] counts: [3, 2, 1]

dataFrameNunique(df, { axis: 1 }) — per row

Count unique values per row (axis=1).

import { DataFrame, dataFrameNunique } from "tsb";

const df = DataFrame.fromColumns({
  a: [1, 2, 2],
  b: [1, 3, 4],
  c: [1, 2, 5],
});

const result = dataFrameNunique(df, { axis: 1 });
console.log(result.values); // [1, 3, 3]
// row 0: {1,1,1} → 1 unique
// row 1: {2,3,2} → 2 unique
// row 2: {2,4,5} → 3 unique
[1, 2, 3]

Missing-value handling

FunctionDefault dropnanull / undefined / NaN treated as
unique()falseOne distinct category (included)
nunique()trueExcluded from count
dataFrameNunique()trueExcluded from count
null, undefined, and NaN are all normalised to the same missing sentinel before de-duplication — consistent with pandas' NaN treatment.

API Reference

// unique
unique(series: Series<Scalar>, options?: UniqueOptions): Series<Scalar>
interface UniqueOptions {
  dropna?: boolean; // default false — include missing values
}

// nunique
nunique(series: Series<Scalar>, options?: NuniqOptions): number
interface NuniqOptions {
  dropna?: boolean; // default true — exclude missing values
}

// dataFrameNunique
dataFrameNunique(df: DataFrame, options?: DataFrameNuniqOptions): Series<number>
interface DataFrameNuniqOptions extends NuniqOptions {
  axis?: 0 | 1 | "index" | "columns"; // default 0 — per-column
  dropna?: boolean;                    // default true
}
