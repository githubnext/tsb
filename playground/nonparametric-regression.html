Nonparametric Tests & OLS/WLS Regression

Rank-based tests that don't assume normality: kruskalWallis, friedmanTest, mannWhitneyU, wilcoxonSigned. Plus olsRegress and wlsRegress for linear regression with full diagnostics.

Kruskal-Wallis H test

Nonparametric one-way ANOVA. Tests whether multiple independent groups come from the same distribution. Uses chi-squared approximation with tie correction. 


    


    
Friedman test

    

      Nonparametric repeated-measures ANOVA. Ranks within each block (row) and tests
      whether column (treatment) rank sums differ.
    

    

      
      
      

    


    
Mann-Whitney U test

    

      Nonparametric two-sample test (Wilcoxon rank-sum). Tests whether two independent
      samples come from the same distribution.
    

    

      
      
      

    


    
Wilcoxon signed-rank test

    

      Nonparametric paired-sample test. Tests whether matched pairs have the same
      distribution (alternative to paired t-test).
    

    

      
      
      

    


    
OLS linear regression

    

      Ordinary Least Squares: fits β in y = Xβ. Include a column of ones
      for an intercept term. Returns coefficients, standard errors, t-stats, p-values,
      confidence intervals, R², F-statistic.
    

    

      
      
      

    


    
WLS regression — down-weighting outliers

    

      Weighted Least Squares: assign lower weight to unreliable observations.
      Equivalent to scaling the system by √w before fitting.
    

    

      
      
      

    


    
Clipboard I/O
