API reference

      

        gaussianKDE(data, options?) — factory; returns a
        GaussianKDE instance.
      

      

        Bandwidth methods:
        silverman (default)  
        scott  
        number (factor × σ)
      

      
Key methods on GaussianKDE:

      
    
        
  • pdf(x) — density at a single point
    • 
        
  • evaluate(points) — density at an array of points
    • 
        
  • logPdf(x) / logpdf(points) — log-density
    • 
        
  • integrate(low, high) — probability mass in interval
    • 
        
  • cdf(x) — cumulative probability up to x
    • 
        
  • integrateFull() — total mass (≈ 1)
    • 
        
  • integrateGaussian(other) — analytic cross-integral
    • 
        
  • resample(size, seed?) — draw samples
    • 
        
  • factor — bandwidth h (kernel σ)
    • 
        
  • covariance — h² (kernel variance)
    • 
        
  • neff — effective sample size
    •
