📊 Contingency Table Analysis scipy.stats.contingency

Association and effect-size measures for contingency tables, mirroring scipy.stats.contingency. Implemented from scratch with no external dependencies.

  • expectedFreq(observed) — expected frequencies under independence
  • relativeRisk(observed) — relative risk (risk ratio) with CI
  • oddsRatio(observed) — odds ratio with CI (Woolf method)
  • association(observed, method) — Cramér's V, phi, contingency C, Tschuprow's T

1 · Expected Frequencies

Under independence, each cell's expected count is
E[i,j] = rowTotal[i] × colTotal[j] / grandTotal.

Press Run to see expected frequencies.

2 · Relative Risk (Risk Ratio)

For a 2×2 table [[a, b], [c, d]]:
RR = (a / (a+b)) / (c / (c+d))

Press Run to compute relative risk.

3 · Odds Ratio

For a 2×2 table [[a, b], [c, d]]:
OR = (a × d) / (b × c)

Press Run to compute odds ratio.

4 · Association Strength

Measures of association derived from the chi-square statistic. All return values in [0, 1] (phi may exceed 1 for tables larger than 2×2).

Press Run to compare association measures.

5 · Real-World Example: Clinical Trial

A drug trial: 200 patients treated (40 recovered), 200 control (25 recovered).

Press Analyse for full summary.
