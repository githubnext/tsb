📈 pct_change

pctChange(series, periods) computes the element-wise percentage change (relative change) between values separated by periods positions. Mirrors pandas.Series.pct_change() and pandas.DataFrame.pct_change(). 

// result[i] = (values[i] - values[i - periods]) / abs(values[i - periods])

import { Series, pctChange, dataFramePctChange } from "tsb";

const prices = new Series({ data: [100, 110, 121, 108.9], name: "price" });
pctChange(prices).values;
//  [NaN, 0.10, 0.10, -0.10]   ← 10% up, 10% up, 10% down

// DataFrame
dataFramePctChange(
  df: DataFrame,
  periods?: number,
  options?: { axis?: 0 | 1 | "index" | "columns" }
): DataFrame

Behaviour notes

  • The first abs(periods) values are NaN (no prior value).
  • Missing values (null, undefined, NaN) propagate as NaN.
  • A prior value of 0 yields ±Infinity — matching pandas.
  • periods=0 yields all NaN (undefined 0/0 change).
  • Negative periods computes a forward look-ahead percentage change.
