between

Element-wise interval membership test — mirrors pandas.Series.between().

1 — Basic range test (inclusive="both")

between(series, left, right) returns a boolean Series that is true wherever the value falls within [left, right] (both endpoints included by default).

import { Series, between } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5], name: "scores" });

const result = between(s, 2, 4);
console.log([...result.values]);  // [false, true, true, true, false]
console.log(result.name);         // "scores"

2 — Inclusive modes

The inclusive option controls which endpoints count:

inclusivecondition
"both" (default)left ≤ x ≤ right
"neither"left < x < right
"left"left ≤ x < right
"right"left < x ≤ right
import { Series, between } from "tsb";

const s = new Series({ data: [2, 3, 4], name: "x" });

console.log([...between(s, 2, 4, { inclusive: "both" }).values]);
// [true, true, true]

console.log([...between(s, 2, 4, { inclusive: "neither" }).values]);
// [false, true, false]

console.log([...between(s, 2, 4, { inclusive: "left" }).values]);
// [true, true, false]

console.log([...between(s, 2, 4, { inclusive: "right" }).values]);
// [false, true, true]

3 — Missing values

null, undefined, and NaN always yield false — they do not fall inside any interval.

import { Series, between } from "tsb";

const s = new Series({ data: [1, null, NaN, 3, undefined] });

console.log([...between(s, 0, 5).values]);
// [true, false, false, true, false]

4 — Filtering rows with between

Combine between with Series.filter (or a DataFrame filter) to select rows in a range:

import { Series, between } from "tsb";

const temps = new Series({
  data: [15, 22, 8, 35, 19, 27],
  name: "temperature_c",
});

// Keep values between 15°C and 25°C (inclusive)
const mask = between(temps, 15, 25);
const comfortable = temps.filter((_v, i) => [...mask.values][i] === true);

console.log([...comfortable.values]);
// [15, 22, 19]

6 — Degenerate interval (left == right)

When left === right, only exact matches pass with inclusive="both"; nothing passes with inclusive="neither".

import { Series, between } from "tsb";

const s = new Series({ data: [1, 2, 3, 4, 5] });

console.log([...between(s, 3, 3).values]);
// [false, false, true, false, false]

console.log([...between(s, 3, 3, { inclusive: "neither" }).values]);
// [false, false, false, false, false]
