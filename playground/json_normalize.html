API Reference

Flatten semi-structured JSON into a flat DataFrame. Mirrors pandas.json_normalize().

jsonNormalize(
  data: JsonObject | readonly JsonObject[],
  options?: JsonNormalizeOptions,
): DataFrame

interface JsonNormalizeOptions {
  recordPath?:   string | readonly string[];   // path to nested record array
  meta?:         ReadonlyArray<string | readonly string[]>;  // parent fields to include
  metaPrefix?:   string;                       // prefix for meta columns (default "")
  recordPrefix?: string;                       // prefix for record columns (default "")
  sep?:          string;                       // key separator (default ".")
  maxLevel?:     number;                       // max flatten depth (default: unlimited)
  errors?:       "raise" | "ignore";           // complex meta values (default "raise")
}
Option Type Default Description
recordPath string | string[] — Path to the nested array of records to unpack. String key or array of keys for nested paths.
meta Array<string | string[]> — Parent-level fields to include as metadata columns in each unpacked row.
metaPrefix string "" Prefix prepended to metadata column names.
recordPrefix string "" Prefix prepended to record column names.
sep string "." Separator used when joining nested key names.
maxLevel number unlimited Maximum nesting depth to flatten. Objects deeper than this level are serialised to JSON strings.
errors "raise" | "ignore" "raise" Behaviour when a meta path yields a complex value: "raise" serialises it, "ignore" uses null.
